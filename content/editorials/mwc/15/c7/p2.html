Thief in the Night Editorial mwc15c7p2

atarw

Use a Prefix Sum Array to store the number of CDs for each floor. Then, simply query for the sum of CDs on the given floor and between the given range.

Time Complexity

\(\mathcal{O}(FR + Q)\), the time complexity of generating the prefix sum array for each floor \(F\) as well as performing each query in \(\mathcal{O}(1)\) time.
