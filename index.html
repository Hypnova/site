Features

The following are some planned features of the site.

  • News - website updates on the home page
  • Learn - lessons about data structures and algorithms
  • Ranks - a leaderboard based on dmoj activity
  • Editorials - solutions/editorials to relevant contests (CCC, MWC, JDCC)
  • Online Judge - an online judge for use in on-site contests
  • Calendar - a calendar with planned events
  • Resources - a list of good compilers, sites, etc.
  • About - site information
