Memoization

Memoization is a technique that saves processing power by caching function call results to avoid reprocessing. Memoization is often used to speed up recursion.

In the fibonacci recursive function example, the the function is called with the same parameter many times. Thus, the results will be stored in an array cache.

The following is the unoptimized fibonacci function. 


    static int fib(int n) {
        if(n <= 2) return 1;
        return fib(n - 1) + fib(n - 2);
    }

To optimize the function, a global array cache, initialized to the maximum size of \(n\) of -1 is created. The function will then be as follows: 


    static int fib(int n) {
        if(n <= 2) return 1;
        if(cache[n] != -1) return cache[n];
        return cache[n] = fib(n - 1) + fib(n - 2);
    }

Base cases may be hardcoded into the cache array or in the function.
