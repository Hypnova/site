Genetic Engineering mhc16g3

Samantha Unger, Sarah Goldfarb

Alice is a mad scientist who is interested in genetically modified humans. She knows this is unethical, but they decide to attempt it anyways (Mwahahaha). She imagines creating humans who are toeless, single-eared, are constantly smiling with no eyes, along with many other configurations. A scientist named Tony had found a way to edit chunks of the genetic code of a human, so that these changes can take place.

They take the genetic code of a preexisting human (it’s very sloppy and has both upper and case letters!), and swap out certain sections to make the alterations that they’d like. They make the alterations and inject the genetic code back into the human!

Input

The first line will contain the alien genome sequence (its length is \(L\) in characters, such that \(0\le L \le 2000\)). This may be in uppercase, lowercase, or a mix. We do not care if they are upper or lower; we only care about the letters. The next line contains the first sequence of letter(s) that must be replaced, a space, and then the sequence of letter(s) that will replace it. The next line contains the second sequence of letter(s) that must be replaced, a space, and then the sequence of letter(s) that will replace it. The final line contains the first sequence of letter(s) that must be replaced, a space, and then the sequence of letter(s) that will replace it. Keep in mind that once you make the changes from the first line, you use that new genome to create the changes for the second line. The changes made from the second line will be used for the third line’s changes.

The swap string we give you may not occur in the genome, so be careful about going out of bounds.

Also, there is no guarantee that this/these letter(s) even exist in the genome sequence. So it may not even be found.

Output

You must output the new genome, in only uppercase letters.

Sample Input

AgCtTAACGctggcaATaGCAt
ag gt
aA GtatA
ta CC

Sample Output

GTCTTGCCCCCGCTGGCGCCCCTGTCAT

Explanation for Sample Output

We start with this:
AGCTTAACGCTGGCAATAGCAT
Then we sub out all the AG.
GTCTTAACGCTGGCAATGTCAT
We then sub out all the AA.
GTCTTGTATACGCTGGCGTATATGTCAT
We then sub out all the TA. GTCTTGCCCCCGCTGGCGCCCCTGTCAT
