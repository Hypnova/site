Array Mapping

Array Mapping is a technique used to associate a key with a value. Primitive arrays map the key type (int) to a value (the type of the array). In this lesson, we will explore mapping with letters as keys. 

array[0] = 1;

Notice the 0 as the accessing index. This is the key. With arrays, the key is always an int.

However, we can take advantage of the properties of a char. chars are easily casted to an int.

Let's try a problem:

Find the number of occurences of each letter in a word.

We need to map each letter to an index in the array. Let's use a to 0, b to 1... z to 25. Conveniently, ASCII values of letters are in the same order, just with a different starting number. 


        String word = "hello";
        int[] occ = new int[26]; // Occurences
        for(char c: word.toCharArray())
        occ[c - 'a']++; // 'a' for lowercase and 'A' for uppercase

Mapping isn't limited to only letters though, it can be used with more characters. Just use a bigger array. 


        String word = "mcpt.ca";
        int[] occ = new int[256]; // Much more ASCII values
        for(char c: word.toCharArray())
            occ[c]++;

        // Print the occurences of only lowercase letters
        for(char i = 'a'; i <= 'z'; i++)
        System.out.println("The letter " + i + " occurs " + occ[i] + " times");

Array mapping can also be used with other keys such as digits: counting the occurences of digits. Later on, we will learn mapping with other data types, such as double, String and even custom classes.
