Breadwinners Editorial mwc15c6p2

atarw

Use the Sieve of Eratosthenes to generate a sorted list of primes up to the maximum \(C_{i}\) value. Then, for each competitor \(i\), use binary search to find the greatest prime number that is still lower than \(C_{i}\).

Time Complexity

\(\mathcal{O}(C_{max} \log \log C_{max} + N \log C_{max})\), the time complexity of generating the sieve as well as binary searching for each competitor.
