Haunted Messages mhc16j4

Sarah Goldfarb

Christine finds a secret message on the wall of a haunted house. She gives us clues to figure out the message. First she gives 5 letter words to find the message. To find the message, we need to mix the words evenly. For example, hello, light, shoes. This would mix the word like this: hlseihlgolheots. Then she gives numbers to indicate the places in this mixed word where the final message shares a value.

Input

The first line includes the number of 5 letter words that she gave as clues, \(C\) \((2 \le C \le 20)\).

Then the next line contains \(C\) five letter words, each separated by a space.

The following line contains \(L\) \((1 \le L \le 15)\), the length of the final message.

The final line contains \(L\) numbers, the indices that Christine uses to create the final message.

Output

A string, the final message.

Sample Input

3
asdfh lncob qwery
9
12 0 1 1 10 5 8 8 4

Sample Output

halloween

Explanation for Sample Output

The mixed message is: alqsnwdceforhby. This message is used for indices where a starts at 0.
