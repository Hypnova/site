Magical Sets Editorial mwc15c6p3

aurpine

This is a very simple bit manipulation problem. Map the existance of each number to a bit in a 64 bit integer (long), using an array of \(N\) longs to store all the sets.

Then, for each query, iteratively merge (xor) the sets (integers). Then scan through the integer and print the existing elements.

Time Complexity

\(\mathcal{O} (61QM)\)
