Adjacency Matrix

An adjacency matrix is used to store edges of a graph. It is often used for dense graphs and/or graphs with no duplicate edges.

Declaration

The underlying data structure is simply a 2D array initialized in both dimensions to the number of nodes. Increment the size if 1-indexed (which it usually is). The type of the array depends on the graph. Typically, the type is int when representing weighted graphs, but you may use boolean for an unweighted graph. 


    int[][] adj = new int[N + 1][N + 1];

Initialization

adj[a][b] describes the edge between node a and b. For weighted edges, a constant is required to identify that no edge exists directly connecting the two nodes. Depending on the situation, it may be more appropriate to use a very large number (such as 0x3f3f3f3f or 2147483647) or -1 (do not use this if negative edge weights are allowed). Since an edge originally doesn't exist, initialize to this value. 


    for(int[] a : adj)
        Arrays.fill(a, -1);

Adding an Edge

Of course, edges may be added differently depending on the graph.

Weighted Undirected

This is the most common type of graph. Given a b w, where a and b are the two nodes to connect and w is the edge weight. 


    adj[a][b] = adj[b][a] = w;

Weighted Directed

Given a b w, meaning a one way edge from a to b with a weight of w. 


    adj[a][b] = w;

Unweighted Undirected

Given a b, a two way edge between a and b. 


    adj[a][b] = adj[b][a] = true;
    // Or if adj is an int type
    adj[a][b] = adj[b][a] = 1; // You might find another number convenient to use

Unweighted Directed

Given a b, a one way edge between a and b. 


    adj[a][b] = true;
    // Or if adj is an int type
    adj[a][b] = 1; // You might find another number convenient to use

Space Complexity

\(\mathcal O (V^2)\)

Time Complexity

Adding edge

\(\mathcal O (1)\)

Checking If Edge Exists

\(\mathcal O (1)\)

Iterating Adjacent

\(\mathcal O (V)\)
