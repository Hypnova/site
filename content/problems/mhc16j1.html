Patterned Pumpkin Picking mhc16j1

Sarah Goldfarb

Madeline is going pumpkin picking. She visits a pumpkin patch with one row. This special pumpkin patch plants pumpkins very strategically. The first pumpkin is planted where Madeline starts. The second pumpkin is planted \(X\) plots away. Then the following pumpkins are each planted \(X\) plots away from the one before it. Madeline must determine the third pumpkinâ€™s position.

Input

Two lines contain, a single integer, \(A B\), the plot number of the first and second pumpkin respectively.

\(1 \le A \lt B \le 150\)

Output

Output the plot number of the third pumpkin.

Sample Input

4
8

Sample Output

12

Explanation for Sample Output

We start at plot \(4\). Then we go to plot \(8\). \(8 - 4 = 4\). So, \(8 + 4 = 12\).
