Square Formation mhc16g2

Samantha Unger, Sarah Goldfarb

4 friends trying to run around the track in a square formation. Running in a square formation means that as they run, they form a square in relation to each other. They are not very good at this sometimes. Given their coordinate locations, determine if the friends are arranged in a square.

Input

The first line will contain integer \(x\) \((-100 \le x \le 100)\), the \(x\) coordinate of the first person and integer \(y\) \((-100 \le y \le 100)\), the \(y\) coordinate of the first person.

The second line will contain integer \(x\) \((-100 \le x \le 100)\), the \(x\) coordinate of the first person and integer \(y\) \((-100 \le y \le 100)\), the \(y\) coordinate of the second person.

The third line will contain integer \(x\) \((-100 \le x \le 100)\), the \(x\) coordinate of the first person and integer \(y\) \((-100 \le y \le 100)\), the \(y\) coordinate of the third person.

The fourth line will contain integer \(x\) \((-100 \le x \le 100)\), the \(x\) coordinate of the first person and integer \(y\) \((-100 \le y \le 100)\), the \(y\) coordinate of the fourth person.

Keep in mind that the runners are in no particular order!

Output

If the students are arranged in a square, output SQUARE FORMATION.

If the students are not arranged in a square, output IMPROPER FORMATION.

Sample Input

0 0
52 52
0 52
52 0

Sample Output

SQUARE FORMATION
