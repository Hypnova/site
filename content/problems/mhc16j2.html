Pumpkin Carving mhc16j2

Vincent Macri

Vincent wants to have the best pumpkin carving design on his street. To do this, he has planned every important coordinate of his design, as he was taught by Mrs. Krasteva and Ms. Dyke.

Given \(N\) points, output the general direction to travel in to get to the next point. Start at the first point.

Input

The first line will contain the integer \(N\), the number of points.

The next \(N\) lines will each contain two integers: the \(x\) value of the point followed by the \(y\) value of the point.

\(1 \le N \le 1000\)

\(-10000 \le x, y \le 10000\)

Output

Output the general direction to cut in. Each cut should have its own line of output.

The possible directions are: UP, DOWN, LEFT, RIGHT, UPLEFT, UPRIGHT, DOWNLEFT, and DOWNRIGHT.

Occasionally, Vincent will wants a part of the pumpkin to be emphasized. On his plan, he lists the same point multiple times if he wants to stab the pumpkin in that spot. When this happens, output STAB as many times as the plan says to.

Vincent also starts by stabbing the pumpkin, therefore you should have \(N\) lines of output.

Sample Input

8
0 0
0 1
0 0
1 0
2 1
-4 -7
5 5
5 5

Sample Output

STAB
UP
DOWN
RIGHT
UPRIGHT
DOWNLEFT
UPRIGHT
STAB
