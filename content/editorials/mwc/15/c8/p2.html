ASCII Art Editorial mwc15c8p2

aurpine

This problem isn't hard, it's just an implementation problem.

Use a variable for the colour. For every operation, just set the character(s) one by one.

The only slight challenge here is the canvas size. Since the canvas will only be \((0, 0)\) to \((99, 99)\) at most, a char array of 100 by 100 initialized to ' ' is sufficient.

Then four variables for the min/max of the columns and rows reached is required. Initialized to the highest/lowest number. Use these variables when printing out the picture.

Time Complexity

\(\mathcal{O}(QCR)\), queries, columns, rows.
