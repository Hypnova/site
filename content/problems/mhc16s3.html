Symmetric Names mhc16s3

Justin Pu

What are you for Halloween?

The simple question that everyone asks. Instead of answering with what you actually are, it's become a trend to answer with a cool name. A cool name has symmetry (read the same backwards).

Given the current name, find the number of ways that the longest cool name can be constructed from the letters of the current name. Not all the letters need to be be used. Each letter can only be used once, i.e. if the original name has 3 As, then you can use at most 3 As.

Input

The first and only line of input contains a string composed only of uppercase letters.

For \(50\%\) of the points, the length of the name doesn't exceed \(20\) letters. For full points, the length of the name doesn't exceed \(40\) letters.

Output

Output a single integer, the number of ways you can form the longest cool name.

Note: The answer may not fit in a \(32\) bit signed number.

Sample Input 1

GHOST

Sample Output 1

5

Sample Input 2

EEVEE

Sample Output 2

1

Sample Input 3

BARBARIAN

Sample Output 3

18
