Dolls Editorial mwc15c1p3

aurpine

This is a simple search problem. First, count the number of occurances of each price. There are only 100 unique prices, so just make an array to contain the number of occurances. Then, iterate through these 100 prices and record the most and least abundant prices.

Time Complexity

\(\mathcal{O}(N)\), where \(N\) is the number of dolls.
