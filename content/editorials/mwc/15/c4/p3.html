Salt Editorial mwc15c4p3

aurpine

This problem requires coordinate compression as the coordinates can go up to \(10^9\): hashmaps and hashsets Use two hashmaps of hashsets, one for each axis.

For example, yaxis[y] contains the x values of all the pieces of salt with the y-coordinate y.

Two hashmaps are used to answer both type 2 queries by finding the size of the respective hashset. Either hashmap may be used to answer the first query.

Time Complexity

\(\mathcal{O}(Q)\), amortized \(\mathcal O (1)\) for type 1 and both type 2 queries.
