Candyman mhc16s5

Atharva Washimkar

It's the night before Halloween, and you need to decide what you're going to give out to the children who visit your house! After a bit of digging around, you manage to find \(N\) candy boxes, each made by a different company. Each of the candy boxes has \(b_{i}\) candy bars in them.

You decide to make candy care packages, each containing exactly \(C\) candy bars, where each bar is made by a different company. Since you don't want to dissapoint any trick-or-treaters, you want to know the maximum number of candy care packages you can make!

Input

The first line contains two space separated integers \(C\) \((1 \le C \le 20)\) and \(N\) \((C \le N \le 50)\), representing the number of candy bars that go into each care package, and the number of candy boxes you have respectively.

The next line contains \(N\) space separated integers \(b_{i}\) \((1 \le b_{i} \le 10^9)\), specifying the number of candy bars in each box.

Output

Output the maximum number of candy care packages you can make. Note: the number may not fit into an int, use long in Java or long long int in C++.

Sample Input 1


    4 5
    4 4 4 4 4

Sample Output 1

5

Explanation for Sample Output 1

Assuming the 5 different boxes are A, B, C, D and E, shown below are the 5 candy care packages you can make using candy from those boxes.

A B C D
A B C E
A B D E
A C D E
B C D E

Sample Input 2

5 6
1 2 3 4 5 6

Sample Output 2

3

Explanation for Sample Output 2

Assuming the boxes are labelled A to F (as ordered in input), shown below are the 3 candy care packages you can make using candy from those boxes.

B C D E F
B C D E F
A C D E F
